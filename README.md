# Prerequisites
- **Node.js** v21.7.2
- **PostgreSQL** v14.15

# Backend
```
cd backend
```
```
npm install
```
```
npm test
```
```
npm start
```

# Frontend
```
cd frontend
```
```
npm install
```
```
npm run dev
```
